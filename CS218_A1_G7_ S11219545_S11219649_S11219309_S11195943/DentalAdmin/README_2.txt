# Dental Server Application
## Overview
The Delta Dentistry Client Application is a Java-based dental appointment booking
application. It allows clients to sign in, book arrangements and
view arrangement history.
The application connects to a remote backend java app
through RMI (Remote Method Invocation). 

## Table of Contents
- Features
- Usage
- Login Screen
- Admin Main Screen

## Features
 - Admin login functionality
 - add new dentist account
 - update existing dentist account
 - delete a dentist account

## Usage

###Login Screen
After starting the application, admin is shown the Delta Dentistry Login Screen. This screen allows users to log in with their credentials.

###Admin Main Screen
The admin main screen has 4 options, which are to create new dentist account, update an existing dentist account or delete account. The admin can also logout of the system.
