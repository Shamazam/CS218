How To Run Program

1) Start server from DentalServer Package by running the java file that has main program
2) Run the login java file from DentalClient Package to login into system or run the register user java file to create a patient account.
3) Both dentist and patient can login from same login frame.
4) example dentist acc, username: Doc1234, Passowrd: 123456.
5) A new dentist account can be created from the DentalAdmin Package, where admin username is Admin1234 and password is 123456.
6) If any other information is missing, you can get it from the access database located in DentalServer Package.
